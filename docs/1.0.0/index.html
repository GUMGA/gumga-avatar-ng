Descrição

O gumga avatar é um componente para manipulação de uma imagem, sua configuração permite cortar imagens quadradas ou redondas, esse componente te permite criar objetos do tipo GumgaImage ou apenas uma URL.

Requisitos:
  • Font Awesome
  • UI Bootstrap

Instalação

// npm
npm i gumga-avatar-ng --save

// bower
bower install gumga-avatar-ng --save

//Injeção do modulo
angular.module('yourApp', ['gumga.avatar']);

<script src="node_modules/gumga-avatar-ng/dist/gumga-avatar.min.js"></script>
<script src="bower_components/gumga-avatar-ng/dist/gumga-avatar.min.js"></script>

Parâmetros
Parâmetro Descrição Valor Padrão
height Define a altura da imagem em pixel. 170
width Define a largura da imagem em pixel. 170
picker-text Define o texto que irá aparecer no botão de alterar foto Atualizar foto
crop-title Define o texto que irá aparecer no titulo do modal que abre no momento do corte da foto. Criar novo recorte
type Define o tipo da imagem se é quadrada ou redonda. square|circle. square
use-gumga-image Define se vai ser aplicado ao ngModel um objeto do tipo GumgaImage ou uma URL da imagem. true|false false
default-avatar Define uma imagem padrão que é exibida quando o ngModel é nulo.
